Dialvox AI te permite conectar tres tipos diferentes de números telefónicos a tu asistente de IA:

1. **Números Normales (Dedicados)**

   - Rentados directamente desde Dialvox.
   - Se pueden usar tanto para llamadas **entrantes** como **salientes**.
   - Generalmente tienen un costo mensual fijo y permiten manejar múltiples llamadas en paralelo con un solo número.

2. **Integración SIP (VOIP/PBX)**

   - Conecta un número existente de VOIP o PBX mediante SIP.
   - Esta configuración te permite conservar tus números actuales mientras aprovechas la inteligencia de Dialvox AI.
   - Funciona tanto para llamadas **entrantes como salientes**, pero requiere que compartas credenciales SIP o los datos del troncal para configurar la integración.
   - Para enviar una solicitud, ponte en contacto con nuestros asesores.

3. **Caller ID (Solo Salientes)**

   - Ideal si ya tienes un número (como el de tu empresa o personal) y no deseas portarlo ni integrarlo por SIP.
   - Verificas la propiedad (mediante OTP o una llamada breve) para que ese número aparezca como identificador de llamada (Caller ID) en llamadas salientes.
   - **No se pueden recibir llamadas entrantes en ese número** mediante Dialvox AI, ya que solo sirve para mostrar el ID de llamada saliente.
