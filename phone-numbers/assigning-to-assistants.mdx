Después de adquirir un número (o configurar SIP o Caller ID), necesitarás asignarlo a tu asistente de IA. El proceso es el mismo tanto si usas el número para llamadas entrantes como salientes.

## Asignar un Número

1. Ve a **Asistentes** → **(Tu Asistente)** → **Configuración**
2. En **Números de Teléfono**, selecciona un número de tu inventario de Dialvox AI
3. Haz clic en **Guardar** para aplicar los cambios

## Uso del Número

Una vez asignado, el número puede utilizarse para:

- **Entrantes**: Las personas que llamen a ese número llegarán a tu asistente de IA
- **Salientes**: El número aparecerá como Caller ID en campañas salientes
- **Ambos**: El mismo número puede manejar llamadas entrantes y salientes

## Verificación de Número

Para números que no fueron rentados directamente desde Dialvox AI:

- Deberás verificar la propiedad mediante un **código OTP** o una **llamada de verificación**
- Una vez verificado, el número podrá usarse para llamadas entrantes y salientes
- El estado de verificación se mantiene incluso si cambias de asistente

## Pruebas

Antes de ponerlo en producción:

1. Realiza una llamada entrante de prueba para verificar que el asistente responde
2. Ejecuta una llamada saliente de prueba para comprobar que el Caller ID se muestra correctamente
3. Verifica cualquier función específica (transferencias, buzón de voz, etc.)

---

**Consejo:** Siempre prueba tanto la funcionalidad entrante como saliente después de realizar cambios en la asignación de números.
