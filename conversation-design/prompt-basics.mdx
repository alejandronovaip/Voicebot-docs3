Dialvox AI utiliza un **Prompt del Sistema** para guiar cómo tu asistente de IA interactúa con los interlocutores. Redactar un prompt efectivo es clave para mantener conversaciones telefónicas más fluidas y precisas.

## 1. Prompt del Sistema vs. Mensaje Inicial

- **Prompt del Sistema**: Instrucciones generales y contexto. Ejemplo:  
  _"Eres el Bot de Soporte de TechMasters Inc. Saluda con cortesía, solicita el nombre, gestiona problemas técnicos básicos y transfiere la llamada si el problema es complejo."_

- **Mensaje Inicial**: Es la primera frase que el asistente dice, especialmente para llamadas entrantes. Ejemplo:  
  _"¡Gracias por llamar a TechMasters! ¿En qué puedo ayudarte hoy?"_

## 2. Claridad y Estructura

- **Sé Específico**: Indica claramente el propósito principal, el tono y los pasos que debe seguir la IA.
- **Usa Ejemplos**: Muestra diálogos de ejemplo. _"Si el usuario dice que tiene una consulta de facturación, pídele el número de factura."_

## 3. Manejo de Casos "Desconocidos"

- Si un usuario pregunta algo fuera de contexto, define si el asistente debe:
  - Dar una respuesta por defecto: _"No estoy seguro, déjame transferirte…"_
  - Solicitar más detalles.

## 4. Mantenlo Breve y Enfocado

- Un prompt corto suele funcionar mejor; instrucciones demasiado largas pueden hacer que la IA sea más lenta o cometa errores.
- Si tienes contenido extenso, considera usar una base de conocimiento o dividir el texto con encabezados.

---

**Consejo:** Revisa periódicamente las transcripciones para detectar dónde tu IA podría necesitar más o menos instrucciones.
