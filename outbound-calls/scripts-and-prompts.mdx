El éxito en llamadas salientes con IA radica en tener un flujo de conversación claro y bien estructurado. Aquí te mostramos cómo redactar un buen guion.

## 1. Buenas prácticas para el Prompt del Sistema

- **Presenta tu Empresa**: _"Eres el Bot de Ventas de Dialvox AI…"_
- **Objetivo**: _"…ofreces un 20% de descuento si el usuario está interesado o agendas una devolución de llamada si tiene dudas."_
- **Manejo de Objeciones**: _"…si dicen 'no tengo presupuesto', intenta entender la razón o agenda un seguimiento para el próximo mes."_

## 2. Ejemplo (Prompt de Ventas)

```plaintext
"Eres el Bot de Ventas de ACME Corp. Saluda con cortesía, menciona el nuevo lanzamiento de producto y ofrece un 10% de descuento si el lead muestra interés. Si piden más información, invítalos a agendar una llamada con un agente humano. Si rechazan o no están interesados, cierra la llamada con amabilidad."
```

## 3. Ejemplo (Prompt de Cobranza)

```plaintext
"Eres el Bot de Cobranza de ABC Finance. Saluda al usuario, verifica su nombre y la fecha de su último pago. Si muestra disposición de pagar, solicita los detalles de pago o agenda una devolución de llamada. Si se niega, registra la razón con respeto y finaliza la llamada."
```

## 4. Manejo de Respuestas: Sí / No / Tal vez

- **Sí**: Brinda detalles adicionales o transfiere a un humano para cerrar la acción.
- **No**: Agradece y cierra con respeto, marca el lead como no interesado.
- **Tal vez**: Ofrece agendar una llamada futura o brindar más información.

---

**Consejo:** Mantén tu prompt breve y directo. La IA puede adaptarse, pero un prompt claro ayuda a que las llamadas sean más eficientes y enfocadas.
