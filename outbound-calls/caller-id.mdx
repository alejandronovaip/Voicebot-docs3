---

Cuando lanzas una campaña de llamadas salientes, puedes elegir qué número aparecerá en el teléfono del destinatario.

## 1. Número Dedicado

- Si has rentado un número local o una línea gratuita (toll-free), puedes seleccionarlo como “Número telefónico” en la configuración del asistente.
- Ideal para mantener coherencia de marca o si también recibes llamadas entrantes en esa línea.
- **No se requiere verificación OTP** para números rentados directamente a través de Dialvox AI.

## 2. Número Existente Verificado (Solo Caller ID)

- Si deseas que aparezca tu número personal o el de tu empresa, puedes verificarlo mediante un **código OTP** o una llamada breve.
- No se podrán recibir llamadas entrantes con este número, pero el destinatario verá tu número habitual en el identificador de llamadas.
- **Nota**: La verificación OTP solo se requiere cuando usas tu propio número existente como Caller ID.

## 3. Integración SIP

- Si usas una PBX o una plataforma VOIP, puedes pasar un identificador de llamada personalizado desde tu sistema existente.
- Asegúrate de que tu troncal o proveedor permita especificar el Caller ID saliente.

## 4. Reglas Específicas por País

- Algunos países no permiten usar como Caller ID números que no te pertenecen.
- Asegúrate siempre de tener el derecho a presentar ese número para evitar problemas legales o que tus llamadas sean marcadas como spam.

---

**Consejo:** Haz una llamada de prueba a tu propio teléfono o al de un colega para verificar que el identificador de llamadas se muestra correctamente antes de lanzar una campaña a gran escala.
