Si necesitas personalización más allá de los bloques sin código, Dialvox AI ofrece soporte para **webhooks** y una **API REST**.

## 1. Webhooks

- **Cuándo Usarlos**: Dispara eventos en sistemas externos en cuanto algo sucede en Dialvox AI (por ejemplo, “Llamada Finalizada”).
- **Configuración**: Define una URL de webhook en la configuración del asistente, campaña o dentro de la plataforma de automatización.
- **Payload**: Generalmente incluye el ID de llamada, transcripción, resultado de la llamada y cualquier dato recogido del usuario.

## 2. API REST

- **Funciones Principales**:

  1. **Crear/Actualizar Leads**
  2. **Gestionar Campañas**
  3. **Consultar llamadas y estadísticas**

- **Autenticación y Límites**: Utiliza tokens de API. Consulta la [documentación de API](/api-reference/introduction) para detalles sobre los endpoints y cuotas de uso.

## 3. Casos de Uso Comunes

- **Sincronización en Tiempo Real con CRM**: Enviar los datos de cada llamada directamente a tu CRM.
- **Dashboards Personalizados**: Extraer información de llamadas para alimentar tus propios paneles de analítica.

---

**Nota:** La documentación oficial de la API está disponible en una sección dedicada o desde tu panel de control de Dialvox AI. Asegúrate siempre de proteger tus tokens y realizar pruebas en un entorno de desarrollo.
