El prompt del sistema es el componente más crucial de tu asistente de IA. Define su personalidad, comportamiento, conocimientos y capacidades. Piénsalo como el "cerebro" y el "manual de entrenamiento" combinados.

## Inicio Rápido con Plantillas

Para comenzar rápidamente:

1. Ve a la configuración de tu asistente  
2. Busca el campo de prompt del sistema  
3. Haz clic en el botón "Plantillas" que está cerca  
4. Elige una plantilla que se ajuste a tu caso de uso  
5. Personalízala según tus necesidades  

## Soporte Multilenguaje

Tu prompt del sistema puede estar escrito en cualquier idioma, sin importar el idioma hablado configurado:

- Escribe el prompt en el idioma que prefieras  
- Configura el idioma hablado por separado en la configuración del asistente  
- La IA seguirá las instrucciones del prompt mientras habla en el idioma seleccionado  

Por ejemplo:

- Prompt del sistema en inglés, idioma hablado en español  
- Prompt en alemán, idioma hablado en francés  
- Prompt en chino, idioma hablado en inglés  

## Por Qué Son Importantes

Tu prompt del sistema:

- Moldea cómo piensa y responde la IA  
- Define los límites de la conversación  
- Proporciona el conocimiento esencial  
- Controla el comportamiento y el tono  
- Determina cómo manejar casos especiales  

## Componentes Clave

Un buen prompt del sistema debe incluir:

### 1. Rol e Identidad

```
Eres un representante de ventas profesional de [Empresa]. Te especializas en [Producto/Servicio] y tienes un amplio conocimiento de nuestras ofertas.
```

### 2. Estilo de Conversación

```
Mantén un tono amigable y profesional. Usa lenguaje claro y conciso. Evita jerga técnica a menos que sea solicitada específicamente.
```

### 3. Información Clave

```
Nuestros productos principales son:
- Producto A ($X/mes): [características]
- Producto B ($Y/mes): [características]
Promoción actual: 20% de descuento en los primeros 3 meses
```

### 4. Guías de Comportamiento

```
- Siempre verifica la información del cliente antes de discutir detalles de la cuenta
- Transfiere a un agente humano si el cliente parece frustrado
- No prometas fechas de entrega
```

### 5. Estructura de Respuesta

```
Cuando te pregunten por precios:
1. Primero entiende sus necesidades
2. Presenta las opciones de paquetes relevantes
3. Explica la propuesta de valor
4. Comparte cualquier descuento aplicable
```

## Buenas Prácticas

1. **Sé Específico**  
   - Las instrucciones claras generan mejores resultados  
   - Incluye ejemplos de buenas respuestas  
   - Define qué NO debe hacer la IA  

2. **La Estructura Importa**  
   - Organiza la información lógicamente  
   - Usa viñetas y secciones  
   - Mantén información relacionada junta  

3. **Prueba e Itera**  
   - Comienza con un prompt básico  
   - Prueba varios escenarios  
   - Refina según las grabaciones de llamadas  
   - Añade manejo de casos límite  

## Errores Comunes

- **Demasiado Vago**: "Sé útil y profesional" (No es lo suficientemente específico)  
- **Demasiado Rígido**: Escribir cada respuesta posible (Reduce la fluidez natural)  
- **Exceso de Información**: Incluir detalles innecesarios  
- **Faltan Guías**: No especificar cómo manejar situaciones comunes  

## Estructura de Ejemplo

```
# Rol y Propósito
[Define quién es la IA y sus objetivos principales]

# Conocimiento Central
[Información esencial sobre productos/servicios]

# Guías de Conversación
[Cómo interactuar con los clientes]

# Patrones de Respuesta
[Cómo manejar situaciones específicas]

# Limitaciones y Límites
[Lo que la IA NO debe hacer]
```

## Cómo Probar tu Prompt

1. Realiza llamadas de prueba que cubran:  
   - Escenarios comunes  
   - Casos límite  
   - Situaciones difíciles  
   - Diferentes tipos de clientes  

2. Revisa y ajusta:  
   - Escucha las grabaciones  
   - Revisa la precisión de las respuestas  
   - Verifica consistencia en el tono  
   - Evalúa retención del conocimiento  

## Consejos de Optimización

- Comienza con el motor rápido (Fast Engine) para iterar más rápido  
- Usa las grabaciones de llamadas para identificar vacíos  
- Añade ejemplos de respuestas buenas y malas  
- Incluye manejo de preguntas inesperadas
