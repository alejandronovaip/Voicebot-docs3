---
title: "Crear lead"
api: "POST /user/lead"
description: "Crear un nuevo lead en el sistema Dialvox AI"
---

Este endpoint permite crear un nuevo lead en el sistema Dialvox AI.

### Cuerpo de la solicitud

<ParamField body="phone_number" type="string" required>
  El número de teléfono del lead en formato E.164 (por ejemplo, +1234567890)
</ParamField>

<ParamField body="campaign_id" type="integer" required>
  El ID de la campaña para la cual se crea el lead
</ParamField>

<ParamField body="variables" type="array">
  Las variables que se pasarán al lead
</ParamField>

<ParamField body="allow_dupplicate" type="boolean">
  Indica si se permiten leads duplicados en una campaña
</ParamField>

### Campos de respuesta

<ResponseField name="message" type="string">
  El mensaje de la respuesta
</ResponseField>

<ResponseField name="data" type="object">
  Los datos de la respuesta
</ResponseField>

<ResponseField name="data.id" type="string">
  El ID del lead
</ResponseField>

<RequestExample>

```json Request
{
  "phone_number": "+1234567890",
  "campaign_id": 1,
  "variables": [
    {
      "customer_name": "John Doe",
      "email": "john.doe@example.com"
    }
  ],
  "allow_dupplicate": false
}
