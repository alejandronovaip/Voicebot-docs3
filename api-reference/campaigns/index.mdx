---
title: "Listar campañas"
api: "GET /user/campaigns"
description: "Lista todas las campañas"
---

Este endpoint permite listar todas las campañas.

### Campos de respuesta

<ResponseField name="campaigns" type="array">
  <Expandable title="propiedades">
    <ResponseField name="id" type="string">
      El ID de la campaña
    </ResponseField>

    <ResponseField name="name" type="string">
      El nombre de la campaña
    </ResponseField>

    <ResponseField name="created_at" type="string">
      La fecha de creación de la campaña
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      La fecha de actualización de la campaña
    </ResponseField>

    <ResponseField name="status" type="string">
      El estado de la campaña
    </ResponseField>

  </Expandable>
</ResponseField>
