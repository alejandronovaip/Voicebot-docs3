La API de Autocalls utiliza claves API para autenticar las solicitudes. Debes incluir tu clave API en el encabezado de cada solicitud que hagas a la API.

## Obtener una clave API

Para obtener una clave API:

1. Inicia sesión en tu cuenta de Dialvox calls en [https://calls.dialvox.io](https://calls.dialvox.io)  
2. Navega a la página "Claves API"  
3. Haz clic en el botón "Crear nueva clave API"  
4. Copia tu nueva clave API y guárdala de forma segura  

<Warning>
  Mantén tu clave API confidencial y segura. No la compartas públicamente ni la incluyas en sistemas de control de versiones.
</Warning>

## Usar la clave API

Incluye tu clave API en el encabezado `Authorization` de cada solicitud:

```
Authorization: Bearer YOUR_API_KEY
```

Reemplaza `YOUR_API_KEY` con tu clave API real.

## Ejemplo de solicitud

Aquí tienes un ejemplo de cómo incluir la clave API en una solicitud cURL:

```bash
curl -X GET "https://call.dialvox.io/api/user/campaigns" \
     -H "Authorization: Bearer TU_CLAVE_API"
```

Reemplaza YOUR_API_KEY con tu clave API real al realizar solicitudes.
